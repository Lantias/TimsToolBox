Tools & Websites Glossary

A growing list of useful resources, with short descriptions and links. I added some notes in a few places, if you have suggestions or want to share some experiances feel free to reach out!

"Like this!"~ Tim

Glossary

Notion
Workspace for notes, docs, and lightweight databases. Great for personal knowledge bases.
tags: notes, productivity, docs
GitHub
Code hosting with issues, pull requests, and CI/CD integrations.
tags: code, version control, collaboration
DrawIO
Free diagramming tool for creating flowcharts and visual diagrams. Recommended for creating network topologies.
tags: drawing, diagrams, network, chart
Google Translate
Free translator for text, speech, images, and entire web pages across 100+ languages; includes pronunciation and a phrasebook. Not the most accurate but highly versetile.
tags: translation, language, productivity
DeepL
Neural-network translator known for high-quality results; supports document translation, custom glossaries, and tone options.
tags: translation, language, productivity
Sejda PDF Editor
Online PDF editor to edit text and images, annotate and sign, and quickly merge/split/compress files; auto-deletes uploads for privacy.
tags: pdf, documents, editing, productivity
Smallpdf
Suite of PDF tools to compress, convert (PDF ↔ Word/Excel/PPT), edit, e-sign, merge/split, and protect files; available on web and desktop.
tags: pdf, documents, conversion, editing, e-signature, productivity
RegExr
Interactive regular expression editor with real-time highlighting and explanations, a built-in cheatsheet, and a community pattern library; test text and share links.
tags: regex, developer tools, testing, text
TextMechanic
Collection of bulk text tools: find/replace, remove duplicates, sort lines, add/remove line breaks, case convert, trim/clean, and more.
tags: text, utilities, formatting, productivity
ToHTML
Online converters for text and code: text→HTML, HTML→text, code→HTML with syntax highlighting, and CSV/JSON→HTML tables; includes HTML encoder/decoder.
tags: html, conversion, formatting, developer tools, text
VirusTotal
Multi-engine scanner for files, URLs, and hashes; checks against dozens of AV engines and sandboxes, with detection details, behavior analysis, and community reports.
"Please take care to not publish personal information"~ Tim
tags: security, malware, analysis, files, urls
Squoosh
Browser-based image compressor and converter with side-by-side preview; supports AVIF, WebP, MozJPEG, and OxiPNG, adjustable quality/resize, and works offline as a PWA.
tags: images, compression, optimization, conversion, web
SS64
Command-line reference and examples for Windows CMD, PowerShell, Bash, and macOS; concise syntax, switches, and scripting notes.
tags: cli, reference, shell, windows, linux, macos, scripting
PDF24
Suite of PDF tools to create, merge/split, compress, convert, edit/annotate, OCR scans, and e-sign; available online and as a free Windows desktop app with on-device processing.
tags: pdf, documents, editing, conversion, ocr, compression, e-signature
Visual Studio Code
Free, extensible code editor for Windows, macOS, and Linux; built-in IntelliSense, debugging, Git integration, terminal, and a rich extensions marketplace (Remote/WSL/Containers supported).
tags: editor, code, ide, extensions, debugging, git, cross-platform
Wikipedia
Free, multilingual encyclopedia written collaboratively by volunteers; a broad reference with citations, summaries, and links to primary sources.
"Make sure to check your sources!"~ Tim
tags: encyclopedia, reference, knowledge, research
